Decentralised Discovery Link
===================================

Decentralised Discovery Link is a project that implements a decentralised protocol designed specifically for data discovery in healthcare. It aims to provide developers and software a way to quickly setup such protocol, integrate with existing systems, and experiment on new ideas.

.. note::

    This project is under active development. API may change without prior notice. Do NOT upgrade without reading the release notes first, especially between major versions.
