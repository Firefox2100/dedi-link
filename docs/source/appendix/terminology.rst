==========================
Terminologies and Concepts
==========================

Some terminologies and concepts used in this project may have different meanings in other contexts. To avoid confusion and ambiguity, this section provides clearer definitions of the terms used in this document.

* **Node**: A node in this project refers to an instance of the system; or more specifically, a back end implementation of the library or its protocol. It's considered a fully configured, self contained unit of functions that will run autonomously with minimal human intervention on the network functions. Some decisions still require human input though.

* **Network**: A network in this project refers to a logical network, containing one or more nodes. This is an abstract model built above layer 7 of OSI model, and should not be confused with the Internet, network connectivity or similar concepts.
