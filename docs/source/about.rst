==================
About the project
==================

This project started as my final year project for my MSc degree in University of Leicester. At the time, I was designing a new version of a data discovery system named Cafe Variome. During the development, I decided to explore a new approach of fully decentralised data discovery protocol, and made the prototype for my own final year project. This library contains the cleaned up, refactored and improved version of the prototype.
